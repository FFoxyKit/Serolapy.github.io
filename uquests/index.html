Предметы

Стандартные предметы CatWar
Подсказка При выборе этого типа распределения предметов все ID предметов в квесте будут равны ID предметов в игре CatWar. То есть если предмет в CatWar имеет ID = 1, то в квесте его ID = #1. Для поиска предметов советую воспользоваться этой моей страницей: Коллекция предметов CatWar.
Полезно, если Вы не добавляете новые предметы в свой квест.

Подсказка При выборе этого типа распределения предметов Вы сами будете создавать предметы, нужные именно Вам. ID созданных предметов будут указаны при создании.
Полезно, если Вы хотите добавить свои предметы, которых нет в CatWar.
